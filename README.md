# Ball Breaker Codex

A neon brick breaker game built with plain HTML, CSS, and JavaScript. Survive endless waves, break tougher bricks, trigger bomb cascades, catch falling boosts, and keep the ball in play.

## Power-ups

- `G` grows the paddle for 15 seconds.
- `S` arms a shield across the bottom of the arena. It blocks one missed ball.
- `B` increases the ball size for 15 seconds.
- `K` makes the paddle sticky so you can aim the next launch.
- `P` phases the ball through bricks for 10 seconds.
- `T` decreases the ball size for 10 seconds.
- `N` decreases the paddle size for 8 seconds. Avoid it.

## Play locally

Open `index.html` or serve the directory with:

```sh
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.
